# Backend for WarMindAI
# This is the backend service for AI-powered person detection and danger assessment.

## Features
- Person detection using OpenCV
- Facial expression analysis
- Object detection for carried items
- Behavior analysis
- Danger level assessment

## Setup
1. Install dependencies: pip install -r requirements.txt
2. Run the server: python app.py
3. Test with: python test.py
